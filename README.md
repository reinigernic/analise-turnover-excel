# analise-turnover-excel
Análise de turnover em base de dados de RH utilizando Excel, com foco na identificação de fatores associados à rotatividade de colaboradores.

## Contexto
A rotatividade de colaboradores (turnover) é um desafio relevante para a gestão de pessoas, impactando custos, produtividade e retenção de talentos. Este projeto explora como a análise de dados pode apoiar a identificação de fatores associados à saída de colaboradores.

## Objetivo
Analisar uma base de dados de Recursos Humanos com o objetivo de identificar os principais fatores associados ao turnover e gerar insights que possam apoiar estratégias de retenção.

## Metodologia
- Análise exploratória de dados (EDA)
- Aplicação da métrica **Information Value (IV)** para seleção das variáveis com maior associação ao turnover
- Análise comparativa entre grupos de turnover e não-turnover

## Principais Insights
- O turnover total identificado foi de **16%** (237 desligamentos em um universo de 1.470 colaboradores)
- A maior concentração de desligamentos ocorre nos primeiros anos de empresa
- Faixas salariais mais baixas e a realização de horas extras apresentam forte associação com o turnover
- Incentivos de longo prazo, como participação acionária, contribuem para a retenção
